## Commands

- `npm run dev` — start the dev server (http://localhost:3000)
- `npm run build` — production build
- `npm run start` — serve the production build
- `npm run lint` — run ESLint (flat config; the `lint` script is bare `eslint`, so pass paths/flags directly, e.g. `npm run lint -- src/app/page.tsx`)
